# go-cli-template

Minimal Go CLI boilerplate/template with zero dependencies

## Features

- minimal CLI implementation
- CI/CD
- golangci-lint
- go test
- goreleaser
- Dependabot
- CodeQL Analysis (Go)

## How to use

1. fork/template this repository
2. replace `midzer` to your username
3. start hacking
